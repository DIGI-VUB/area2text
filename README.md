# area2text

Tool to match image regions with transcriptions

- Run locally `shiny::runGitHub(repo = 'area2text', username = 'DIGI-VUB')`
- Available at https://apps.digitagger.org/digi/getuigenissen2-area2text/ (login with VUB account)
